# Example - Bring your own plan

Use this when the planning work already exists and you want Plan
Enforcer to supply control, accountability, and closure truth.

## Import the plan

```bash
plan-enforcer import docs/plans/external-plan.md
plan-enforcer review docs/plans/external-plan.md
claude
```

Then:

```text
Execute docs/plans/external-plan.md
```

## Why this matters

You keep the existing planner:

- GSD
- Superpowers
- your own markdown plan

But execution still lands inside the same ledger, verify, audit, and
report path.
