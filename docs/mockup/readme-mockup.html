02WHAT THIS MAKES PROVABLE

Plan Enforcer targets the moments AI coding gets slippery: ask narrowing, mid-flight plan mutation, stale-context resumes, and "done" declared before repo truth catches up. Those failure modes stop being invisible — each one has a file that records it.
